## Local Interpretation

These sources support a research direction, not a biological equivalence claim.
The repo should avoid claiming that infant dopamine, attention, or social
learning is directly reproduced. The first implementation only tests whether
AD-first and DA-delayed training can improve small RL sample efficiency.
